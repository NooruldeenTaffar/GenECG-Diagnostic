# GenECG Diagnostic Tool

This project is a tool for diagnosing ECGs using the GenECG dataset.

## Setup

1.  Install the required packages:
    ```
    pip install -r requirements.txt
    ```
2.  Add your Hugging Face token to a `.env` file in the root of the project:
    ```
    HUGGING_FACE_TOKEN="your_token_here"
    ```
3.  Run the download script:
    ```
    python src/Data_pipeline/download_data.py
    ```
